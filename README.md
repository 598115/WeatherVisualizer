# WeatherVisualizer

## Libraries Required for Running Project

**Gradio**:  
`pip install gradio`

**OpenAI**:  
`pip install openai`

## Other Requirements

- An OpenAI/ChatGPT API key assigned to the variable `OPENAI_IMAGE_KEY` in `main/util/apikey.py` (The `apikey.py` file must be created manually).
